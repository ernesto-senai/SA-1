# SA-1
SA 1 – Situação de Aprendizagem

## Inicie o servidor:
\`\`\`bash
node index.js
\`\`\`

## Criar Tarefa
\`\`\`bash
POST /tasks
\`\`\`


## Listar Todas as Tarefas
\`\`\`bash
GET /tasks
\`\`\`

## Buscar Tarefa por ID
\`\`\`bash
GET /tasks/:id
\`\`\`

## Atualizar Tarefa
\`\`\`bash
PUT /tasks/:id
\`\`\`

## Remover Tarefa
\`\`\`bash
DELETE /tasks/:id
\`\`\`
